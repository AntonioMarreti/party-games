---
description: Как обновить версию приложения и сбросить кэш у пользователей
---

Для автоматического обновления версии активов (CSS/JS) используйте скрипт `bump-version.js`. Он считывает количество коммитов в Git и записывает новую версию в `layout/version.php`.

### Как запустить:

1. Откройте терминал в корне проекта.
2. Выполните команду:
```bash
node tools/bump-version.js
```

### Что произойдет:
- Скрипт вычислит новый номер сборки на основе Git (или таймстампа).
- Обновится файл `layout/version.php` (единый источник истины).
- Все ссылки в `index.php` (и в старом `index.html` для совместимости) автоматически получат новый параметр версионности (например, `?v=2326`).
- У всех пользователей при следующем входе загрузятся самые свежие версии файлов, предотвращая баги из-за кэша.
